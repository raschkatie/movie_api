Movie API

This movie API's goals and functions will include:

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies. Example:
{
"Title": "The Lord of the Rings: The Fellowship of the Ring",
"Director": "6626bceaaf9c355c6d117b7c",
"Genre": "6626c157af9c355c6d117b85"
},
{
"Title": "Interstellar",
"Director": "6626bdeeaf9c355c6d117b7e",
"Genre": "6626c17aaf9c355c6d117b87"
},
Return data about a single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie. Example:
{
"Title": "The Lord of the Rings: The Fellowship of the Ring",
"Director": "6626bceaaf9c355c6d117b7c",
"Genre": "6626c157af9c355c6d117b85"
}
Return data about a genre by name/title /movies/genre/[Genre] GET None A JSON object holding data about a specific genre. Example:
{
"_id": "6626c194af9c355c6d117b88",
"Name": "Horror",
"Description": "Horror is a genre of fiction that is intended to disturb, frighten or scare."
}
Return data about a director by name /movies/director/[Director's Name] GET None A JSON object holding information about a specific director. Example:
{
"_id": "6626bceaaf9c355c6d117b7c",
"Name": "Peter Jackson",
"Bio": "Sir Peter Jackson made history with The Lord of the Rings trilogy, becoming the first person to direct three major feature films simultaneously.",
"Birth": "1961"
}
Allow new users to register /account POST A JSON object about the user being added. Example:
{
"Username": "Katie",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14"
} 		A JSON object holding data about the user that was added. Example:
{
"Username": "Katie",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14",
"id": "12345"
}
Allow users to log in /login POST Key-Value pairs in Parameters or the Body with the user's login information. Example:
Key Value
Username placeholder_name
Password examplepassword
A JSON object holding data about the user, including the JWT token. Example:
{
"Username": "Katie",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14",
"id": "12345"
}
"token": "123456789"

Return information about a single user /account/[Username] GET None A JSON object holding data about the user. Example:
{
"Username": "Katie",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14",
"id": "12345"
}
Allow users to update their user info /account/[Username] PUT A JSON object holding data that the user would like to change. Example:
{
"Username": "Judy"
} 		A JSON object holding data about the user, including the change. Example:
{
"Username": "Judy",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14",
"id": "12345"
}
Allow users to add a movie to their list of favorites /account/[Username]/favorites/[Movie ID] POST None A JSON object holding data about the user, including the updated movie list. Example:
{
"Username": "Katie",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14",
"id": "12345",
"FavoriteMovies": ["6626c559a406dc428b117b7b", "6626c567a406dc428b117b7c"] }
Allow users to remove a movie from their list of favorites /account/[ID]/favorites/[Movie ID] DELETE None A JSON object holding data about the user, including the updated movie list. Example:
{
"Username": "Katie",
"Password": "supersecure",
"Email": "katie@gmail.com",
"Birthday": "1990-12-14",
"id": "12345",
"FavoriteMovies": ["6626c559a406dc428b117b7b"] }
Allow existing users to deregister /account/[ID] DELETE None A text message indicating whether the user was successfully removed.
