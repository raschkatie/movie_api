Movie API

This movie API's goals and functions will include:

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies.
Return data about a single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie. Example:
{
"title": "The Lord of the Rings: The Fellowship of the Ring",
"director": "Peter Jackson",
"genre": "Fantasy"
}
Return data about a genre by name/title /movies/genre/[Genre] GET None A JSON object holding data about movies in a specific genre. Example:
{
"title": "The Lord of the Rings: The Fellowship of the Ring",
"director": "Peter Jackson",
"genre": "Fantasy"
}
Return data about a director by name /movies/director/[Director's Name] GET None A JSON object holding movies from a specific director. Example:
{
"title": "The Lord of the Rings: The Fellowship of the Ring",
"director": "Peter Jackson",
"genre": "Fantasy"
}
Allow new users to register /account POST A JSON object about the user being added. Example:
{
"name": "Katie",
"email": "katie@gmail.com"
} 		A JSON object holding data about the user that was added. Example:
{
"id": "12345"
"name": "Katie",
"email": "katie@gmail.com"
}
Allow users to update their user info /account/[ID] PUT A JSON object holding data that the user would like to change. Example:
{
"name": "Judy"
} 		A JSON object holding data about the user, including the change. Example:
{
"id": "12345"
"name": "Judy",
"email": "katie@gmail.com"
}
Allow users to add a movie to their list of favorites /account/[ID]/favorites POST A JSON object with the movie information to be included. Example:
"title": "Howl's Moving Castle" 		A JSON object with the final list of favorites. Example:
[{
"title": "Howl's Moving Castle",
"director": "Hayao Miyazaki",
"genre": "Adventure"
},
{
"title": "The Lord of the Rings: The Fellowship of the Ring",
"director": "Peter Jackson",
"genre": "Fantasy"
}]
Allow users to remove a movie from their list of favorites /account/[ID]/favorites/[Movie] DELETE None A JSON object with the final list of favorites. Example:
{
"title": "Howl's Moving Castle",
"director": "Hayao Miyazaki",
"genre": "Adventure"
}
Allow existing users to deregister /account/[ID] DELETE None A text message indicating whether the user was successfully removed
